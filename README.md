# Aplicativo de Agendamento de Serviços

Este é um aplicativo Flutter simples para agendamento de serviços, onde os usuários podem cadastrar pessoas, serviços e agendar horários.

## Funcionalidades

- Cadastro de Pessoas: Os usuários podem adicionar informações sobre clientes, como nome, telefone, e-mail, etc.
- Cadastro de Serviços: Os usuários podem adicionar informações sobre os serviços oferecidos, como nome, descrição, preço, etc.
- Agendamento de Horários: Os usuários podem agendar horários para os serviços, associando-os a clientes específicos e serviços selecionados.
